BOOMBOT by TerrordactylDesigns

Chat and DJ bot for Turntable.fm

Need your own robot DJ? Want to welcome people to your room without typing? Want to auto tweet the current playing song? Want to impress the ladies? You want BOOMBOT.........

Dependencies

Node.js
NPM
ttapi

Optional Requirements

Twitter API account
Last.fm API account

Install

once you have node running:

Create a FB or twitter account for your new bot.
Log in as the bot and use this to get your bots uid, auth token, and your rooms id:
http://alaingilbert.github.com/Turntable-API/bookmarklet.html
Now use the same method to get your personal accounts user id for control of the bot

Create a folder and path into it with your terminal client of choice.

npm install boombot

Rename example.config.json to config.json and open in Sublime text 2 (Or your editor of choice... but really... go get sublime... its great..)
replace the xxxxxxxxxxxxxxxxxxxxxxxx's in the botinfo and roomid sections with your bots auth, bot id, and your own personal user id in the admin section.
replace botname with what you want to call the bot in the botinfo section.

Download scripts from here, and copy the desired ones to the boombot/scripts folder.

path to the saved location (yourfolder/node_modules/boombot):

node bin/boombot

Twitter/Last.fm Support

Sign up for the API accounts you wish to use. Modify the config files with the given API keys.
For Twitter - set tweet to true, fill in your rooms URL. The bot will now auto tweet the current playing song and a link to the room.
For Last.fm - set use to true. The bot will now return artist and song information. Scrobble support coming soon.

Tests

run npm install from the boombot folder.

npm test

DJ Queue

Due to demand I have added a queue system. It was written offhand on the fly, so there may be some issues. If you find any please let me know. This is V1 of the queue so there will be some fixes (such as a true no limit instead of 100) and feature adds to follow.
To set up, just keep the config file setting to true. If you could care less about a queue system just set that to false.

Commands

Download boombots scripts and copy the ones you want into the scripts folder.

Everyone:

  • /lyrics - returns lyrics snippet and link to full lyrics from lyrics wiki
  • /video - returns a link to a youtube video search for the current song
  • /hello - says hello back
  • /boombot - information about the bot and how to acquire one
  • /help - list the commands
  • /rules - list the rooms rules
  • meow - random super troopers meow reference
  • /version - display the bots version number
  • /mods - returns a list of the current mods online in the room
  • /album - return the album the current song is from (requires last.fm setup)
  • /artistinfo - return information about the current artist (requires last.fm setup)
  • /similarartists - return 3 similar artists to the current playing one (requires last.fm setup)
  • /similarsongs - return 3 similar songs to the current playing one (requires last.fm setup)

Queue commands

  • /q - gives information on the current queue settings
  • q - list the current queue
  • q+ - add yourself to the queue
  • q- - remove yourself from the queue
  • /plays - list the play counts of each DJ on stage
  • /settings - see the queues current settings
  • vip <user name> - shift dj to the first position in queue

Administrator/"The Master"

These commands are given as: <botname> <command>

  • djmode - make the bot get on stage and DJ
  • getdown - make the bot get off the stage
  • skip - make the bot skip the song it is playing
  • addsong - add the current playing song to the bots playlist
  • good - remind the bot he's a good boy. Maybe he will kill you last during the robot apocalypse.
  • dance - make the bot upvote **this feature is against the TOS of turntable.fm and is disabled**
  • voodoo up - switch the bot to the voodoo doll avatar (requires 100 DJ points)
  • kitten up - switch the bot to the green kitten avatar (requires 300 DJ points)
  • alien up - switch the bot to the alien avatar (requires 100 DJ points)
  • vampire up - switch the bot to the vampire avatar (requires 100 DJ points)
  • phone up - sets the bots laptop to the iPhone
  • fruit up - sets the bots laptop to Mac
  • nix up - sets the bots laptop to Linux/Ubuntu
  • chrome up - sets the bots laptop to ChromeOS
  • droid up - sets the bots laptop to Android phone

Room Moderator Commands

Queue Control

  • /q on - turn the queue on
  • /q off - turn the queue off
  • /1 - set the song limit to 1
  • /2 - set the song limit to 2
  • /3 - set the song limit to 3
  • /none - set the song limit to 100

Room Administration

  • <botname> autobop engage - turns on bot autovoting (If there is only one voting bot in a room this is allowed)
  • <botname> autobop disengage - turns off bot autovoting
  • <botname> blacklist <user name> - boots user from the room and auto boots on entrance

Moderator Chat

  • Room moderators can PM the bot and it will PM all other moderators in the room, like a private group chat.
